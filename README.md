# Stencilwashcoder Brand

Official brand assets and design system for Stencilwashcoder.

## 🎨 Brand Identity

### Vibe
- **Techy** - Clean lines, monospace aesthetics, terminal-inspired
- **Modern** - Minimal, professional, developer-focused
- **Distinctive** - Memorable mark that works at small sizes

### Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Primary | Electric Blue | `#00D4FF` |
| Secondary | Dark Navy | `#0A0E1F` |
| Accent | Neon Green | `#00FF88` |
| Neutral | Cool Gray | `#8B92A8` |

### Typography
- **Primary**: JetBrains Mono or similar monospace
- **Secondary**: Inter or system sans-serif
