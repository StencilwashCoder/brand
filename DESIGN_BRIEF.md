# Design Brief: Stencilwashcoder Logo

## Overview
Create a distinctive, tech-forward brand mark for the Stencilwashcoder GitHub presence.

## Inspiration Directions

### 1. Terminal/Code Aesthetic
- `>_` prompt symbol integrated
- Brackets `[]` or `{}` as framing
- Cursor blink element
- Monospace letterforms

### 2. Geometric Abstract
- Hexagon grid (honeycomb/structure)
- Circuit board traces
- Connecting nodes/dots
- Angular, precise shapes

### 3. Literal "Stencil" Concept
- Cut-out/overlaid layers
- Spray paint drip effects (subtle)
- Mask/cutout shapes
- Industrial/utilitarian feel

## Requirements

### Logo
- [ ] Works at 32x32 (favicon)
- [ ] Works at 1024x1024 (GitHub profile)
- [ ] Readable in monochrome
- [ ] Has dark mode variant

### Icon
- [ ] Simplified version of logo
- [ ] Square aspect ratio
- [ ] Recognizable at 16x16

### Banner
- [ ] 1280x640 pixels
- [ ] Incorporates logo
- [ ] Space for tagline
- [ ] Works with GitHub's dark theme
